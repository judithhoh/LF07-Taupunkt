\section{Einleitung}

In diesem Projekt soll eine Taupunktlüftung mithilfe eines Raspberry Pi
realisiert werden. Ziel des Projekts ist es, die Luftfeuchtigkeit in einem
Raum zu regulieren, um ein angenehmes Raumklima zu schaffen und
Schimmelbildung vorzubeugen. 

Die Akzeptanzkriterien für das Projekt sind:
\begin{itemize}
    \item Die Temperatur und Luftfeuchtigkeit sollen über zwei Sensoren erfasst werden.
    
    \item Für beide Messpunkte soll jeweils die Taupunkttemperatur berechnet werden.
    
    \item Anhand des Taupunkt-Deltas soll entschieden werden, ob der Lüfter aktiviert wird.
    
    \item Alle erfassten Daten sollen in einer Datenbank gespeichert werden.
    
    \item Die Messdaten sollen auf einer Webseite dargestellt werden.
    
    \item Für den Remote-Zugriff soll eine VNC-Verbindung zum Raspberry Pi eingerichtet werden.
\end{itemize}


Die Nutzung weiterer Komponenten des bereitgestellten JOY-PI,
wie beispielsweise des Displays, ist optional und nicht zwingend erforderlich.

Des Weiteren kann die Webseite um zusätzliche Funktionen erweitert werden,
beispielsweise um die Möglichkeit, die Lüftungssteuerung manuell zu
überschreiben oder historische Messdaten anzuzeigen.
