\section{Einleitung}

Es soll ein Delta Taupunktlüftung per Raspberry Pi b4 realisiert werden.
Delta Taupunktlüftung:
Luft kann mehr Feuchtigkeit aufnehmen je wärmer diese ist.
Taupunkt: abhängig von Temperatur und Luftfeuchtigkeit wird berechnet, wann das Wasser kondensiert (Temperatur).
Luftfeuchtigkeit soll raus => innen 5,6 °C <- außen 15,3 => abkphlen bis außen geringer als innen
2 Senoren => Temperat + Luftfeuchtigkeit => Delta berechen
Lüfter an wenn nach berechung sinnvoll
Daten (Sensor, Ergenbisse) In Datenbank speichern und auf Webside darstellen
