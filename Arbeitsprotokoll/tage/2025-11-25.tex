\subsection{25.11.2025}

\subsubsection{Ziel}
Das Ziel des Tages war es, die ersten Lektionen aus der JOY-PI-Anleitung
durchzuführen, um sich mit der Ansteuerung der Komponenten über Python
vertraut zu machen.

\subsubsection{Durchgeführte Arbeiten}
Bevor die Lektionen durchgeführt werden konnten, musste zunächst die
notwendige Software auf dem JOY-PI installiert werden. Hierzu wurden die
in der Anleitung beschriebenen Schritte durchgeführt.

Dazu gehörten die Installation von Python sowie der benötigten
Bibliotheken für die Sensoren und die Datenbankanbindung.

\subsubsection{Problem 1: Fehler bei der Installation von Python-Bibliotheken}
Während der Installation der Python-Bibliotheken trat die Fehlermeldung
„error: externally-managed-environment“ auf.

\textbf{Lösung}
Nach einer kurzen Recherche im Internet wurde festgestellt, dass dieses
Problem durch die Verwendung einer virtuellen Python-Umgebung gelöst
werden kann.

Daraufhin wurde eine virtuelle Umgebung mit \texttt{venv} erstellt und
die benötigten Bibliotheken innerhalb dieser Umgebung installiert.

\subsubsection{Nächste Schritte}
Für die nächste Woche ist geplant, die restlichen Bibliotheken zu
installieren und anschließend die ersten Lektionen der JOY-PI-Anleitung
durchzuführen.
